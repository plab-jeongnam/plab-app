# plab-app 비개발자 가이드

> "이 레포로 웹사이트 하나 만들어주세요"를 LLM(예: Claude Code / ChatGPT CLI)에게 말하면, LLM이 이 도구를 써서 웹사이트를 대신 만들어줍니다. 아래는 **그 흐름이 부드럽게 굴러가도록** 당신이 미리 알아둘 것들이에요.

## 한 줄 요약

1. AI 채팅창을 엽니다 (Claude Code 또는 ChatGPT CLI).
2. 아래 **마법의 한 줄**을 복사해 붙여넣고, 만들고 싶은 걸 말합니다.
3. AI가 3~4개 쉬운 질문을 던져요. 답만 해주세요.
4. 몇 분 기다리면 웹사이트가 완성되고, 원하면 배포까지 자동으로 끝납니다.

## 미리 준비해둘 것

| 항목 | 설명 | 없어도 될까? |
|------|------|------------|
| **GitHub 계정** | 코드가 저장될 곳 ([여기서 가입](https://github.com/signup)) | ❌ 필수 |

| **AI 채팅 도구** | Claude Code, Cursor, 또는 ChatGPT CLI 등 터미널 명령을 실행할 수 있는 AI | ❌ 필수 |

## 마법의 한 줄 (AI에게 붙여넣으세요)

```
https://github.com/plab-jeongnam/plab-app 이 도구로 웹 프로젝트를
하나 만들어주세요. 필요한 건 설치부터 다 해주시고, 저한테는
꼭 필요한 것만 물어봐 주세요.
```

이게 전부예요. AI가 알아서:
- 적절한 버전의 plab-app을 **자동으로 다운로드**하고
- **필수 도구**(Git, Node.js 등)를 **자동 설치**하고
- 프로젝트를 만들고, GitHub에 올리고, 필요하면 배포까지 해줍니다.

## AI가 물어볼 3~4개 질문

AI는 모든 걸 대신하지만, **취향이 들어가는 선택지 몇 개**는 당신한테 물어봐요. 미리 알아두면 대답이 빨라집니다.

### 1️⃣ 프로젝트 이름이 뭐예요?

- 예: `landing`, `admin`, `dashboard`, `my-project`
- 영문 소문자·숫자·하이픈(-) 만 가능
- 자동으로 앞에 `plab-`이 붙어요 → `plab-landing`

### 2️⃣ 플랩 데이터를 쓰실 건가요? (예/아니오)

- **예**: 플랩의 경기/유저/결제 같은 데이터를 불러오는 코드가 자동 포함
  - ⚠️ 이 경우 **플랩 API 키**가 필요해요. 담당자에게 "플랩 API 키 주세요"라고 요청.
- **아니오**: 독립적인 일반 웹사이트

### 3️⃣ 플랩 API 키 (2번에서 "예"를 고른 경우에만)

- 담당자에게 받은 긴 문자열을 그대로 붙여넣으면 됩니다.
- 못 받았으면 받고 다시 시도해도 돼요 — AI가 친절하게 멈춰줍니다.

### 4️⃣ 리서처만 쓰는 서비스인가요? (예/아니오)

- **예**: Google 로그인 필수 (외부인 접근 차단됨)
- **아니오**: 누구나 볼 수 있는 공개 사이트

## 중간에 "당신이 직접 해야 하는" 순간들

AI가 대부분을 대신하지만, **브라우저에서 로그인**하거나 **비밀번호를 입력**하는 건 본인만 할 수 있어요. 이럴 때 AI가 멈추고 알려줍니다.

### 🔐 GitHub 로그인

- 터미널에 `gh auth login` 비슷한 안내가 뜨면, 브라우저가 자동으로 열려요.
- GitHub 아이디/비밀번호로 로그인하고 "Authorize" 버튼을 누르면 끝.

### 🚀 Vercel 로그인 (배포할 때만)

- GitHub 계정으로 한 번에 가입·로그인 가능.
- 역시 브라우저에서 "Authorize" 누르면 끝.

### 🍺 Homebrew 설치 (macOS 최초 1회)

- 이건 AI가 자동 실행하기엔 위험해서 본인이 직접 명령어를 복사해 실행해야 해요.
- AI가 **그 한 줄을 알려주면** 그대로 터미널에 붙여넣고 Enter.
- macOS 비밀번호를 물어보면 입력 (화면에 안 보여도 정상).

### 🔑 Google OAuth 설정 (리서처 전용 프로젝트를 배포할 때만)

- 배포 후 Google Cloud Console에 "승인된 리디렉션 URI"를 하나 추가해야 해요.
- AI가 **그 URL을 보여주면** 복사해서 [Google Cloud Console](https://console.cloud.google.com/apis/credentials)에 붙여넣기.

```
나: https://github.com/plab-jeongnam/plab-app 이거로 프로젝트 하나
    만들어주세요.

AI: 네, 지금 plab-app 도구를 설치하고 환경을 점검할게요.
    (잠시 후)
    환경 세팅이 끝났어요. 몇 가지 여쭤볼게요.

    1. 프로젝트 이름을 뭘로 할까요? (예: landing, admin)
나: onboarding

AI: 2. 플랩 데이터를 가져다 쓸 예정이신가요?
나: 네

AI: 플랩 API 키가 필요해요. 담당자에게 받으신 키를 알려주세요.
나: plab_live_xxxxxxxxxx

AI: 3. 이 사이트는 리서처만 보는 건가요, 아니면 누구나 볼 수 있나요?
나: 누구나 볼 수 있게요

AI: 좋아요! 프로젝트를 만들고 있어요... (2~3분 소요)
    ✓ 폴더 생성
    ✓ 패키지 설치
    ✓ 빌드 검증
    ✓ GitHub 저장소 생성 및 푸시

    완성됐어요! 이제 배포해볼까요?
나: 네

AI: Vercel에 로그인해 주세요. 브라우저가 열릴 거예요.
    (본인이 브라우저에서 로그인)
    ✓ 배포 완료!
    https://plab-onboarding.vercel.app 에서 확인하실 수 있어요.
```

## 문제가 생기면?

### "어떤 도구가 없대요" 라고 AI가 말할 때

그냥 **"설치해주세요"**라고 답하세요. AI가 알아서 해줘요. Homebrew 하나만 예외 — 그건 본인이 터미널에 복붙.

### "API 키가 필요하대요"

담당자에게 플랩 API 키를 받아서 AI에게 알려주세요. 받을 때까지 잠시 멈췄다 재개해도 괜찮아요.

### "GitHub에 같은 이름의 저장소가 이미 있대요"

프로젝트 이름을 살짝 바꿔서 다시 시도 (예: `landing` → `landing-v2`).

### "배포가 실패했어요"

AI에게 `plab-app status --json` 실행 결과를 물어보라고 하면 어디가 문제인지 알려줘요.

### 뭔가 꼬였을 때 리셋하기

```
AI에게: plab-app reset 명령으로 복구해주세요
```

### 전부 다시 시작하고 싶을 때

해당 프로젝트 폴더를 통째로 삭제하고 처음부터 다시 시작하면 됩니다. GitHub에 만들어진 저장소는 github.com에서 직접 삭제하거나 그냥 두셔도 돼요.

## 완성된 다음에 할 수 있는 것들

AI에게 한국어로 말하면 다 해줘요. 예시:

- **"개발 서버 켜줘"** → 로컬에서 미리보기 (`plab-app dev`)
- **"새로 배포해줘"** → 변경사항을 배포 (`plab-app deploy --prod`)
- **"지금 내 프로젝트들 뭐 있지?"** → 로컬 + GitHub 프로젝트 목록 (`plab-app list`)
- **"GitHub 저장소 열어줘"** → 브라우저 오픈 (`plab-app open`)
- **"배포된 사이트 열어줘"** → 배포 URL 브라우저 오픈 (`plab-app open`)

## 자주 묻는 질문

**Q. 코드를 한 줄도 몰라도 써도 되나요?**
네. AI가 모든 터미널 명령을 대신 실행해요. 당신은 3~4개 질문에 답하고, 가끔 브라우저에서 로그인만 하면 돼요.

**Q. 돈이 드나요?**
plab-app, GitHub(개인 저장소), Vercel(취미용 플랜) 모두 **무료**로 시작 가능. 플랩 API는 회사 계정이니 사내에서 받으세요.

**Q. macOS만 되나요?**
macOS(Apple Silicon/Intel)와 Windows 모두 지원합니다.

**Q. 만든 프로젝트는 내 거예요?**
네, GitHub 계정에 **private 저장소**로 자동 저장돼요. 나만 볼 수 있고, 원하면 팀원에게 공유할 수 있어요.

**Q. AI가 잘못된 일을 하면?**
파괴적인 작업(폴더 삭제, 강제 푸시 등)은 AI가 꼭 먼저 물어봐요. 동의하지 않으면 진행되지 않습니다. 불안하면 "먼저 뭘 할 건지 알려주고 동의 받아줘"라고 말해도 돼요.
